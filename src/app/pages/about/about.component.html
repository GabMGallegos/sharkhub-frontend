Una interfaz moderna para una barbería organizada

SharkHub representa la migración visual de una aplicación de barbería hacia Angular, conservando el estilo Black & White y una estructura clara para el usuario final.

La interfaz se divide en componentes reutilizables para facilitar mantenimiento, escalabilidad y futura conexión con una API desarrollada en TypeScript.

Componentización

Navbar, Footer, Home y páginas internas se separan en componentes Angular.

UI/UX

El diseño conserva la identidad visual del proyecto original.
